PiTank rolling release!

April 1, 2015July 22, 2015 apostal 0 Comments

You can see that pitank.com is in production with only one post in blog and partly configured WordPress… mostly not configured, except security part (thanks to Artem :)).
Be ready it will be changed frequently, will have some bugs and probably some nice features like PiTank hardware/software had during initial development cycle (about a year ago).
I would call it rolling release, because it reminds me the times when ArchLinux was switching from init to systemd, and it was… fun, so be prepared.
By the way Raspberry Pi Tank run on armv6 Arch Linux, so only fresh kernel(Linux alarmpi 3.18.9-1-ARCH, ok, was not updated for a while, hm, for a week?), ruby 2.2.0 (ah, each major ruby update usually gives pain with installing gems, so hope rvm will fit well for me) and no regrets about stable debian.

So you guys are warned, hope we all will have fun!
