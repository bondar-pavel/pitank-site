Received Raspberry Pi 2

July 10, 2015July 22, 2015 apostal 0 Comments BCM2836, RaspberryPi, RaspberryPi2

Hey! New updates from PiTank family!

Recently I received Raspberry Pi 2 and glad to share my happiness with you.

It comes in small box, where you can find instruction and Raspberry Pi 2 itself in antistatic bag. Instruction contains information on 16 languages (all languages I need are covered).

Probaly you already know something about it:

  • It has absolutely the same design as Raspberry Pi model B+, so all cases and expansion boards are compatible!
  • It uses another ARM chip:  900MHz quad-core ARM Cortex-A7 CPU, BCM2836. And it is 6 times faster than previous model!

  • It has 1 Gb RAM, comparing to 512 for previous model

And all other specifications are the same for Raspberry Pi model B+:

  • 4 USB ports
  • 40 GPIO pins
  • Full HDMI port
  • Ethernet port
  • Combined 3.5mm audio jack and composite video
  • Camera interface (CSI)
  • Display interface (DSI)
  • Micro SD card slot

 

I think new power will help for PiTank project to move forward with such thing like computer vision, which require a lot of computing power.
