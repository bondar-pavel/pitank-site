]]> It was probably the first PiTank test drive filmed on the camera. It had no camera built on top of pi, and no cool WebUI interface, only console, only hardcore. But base functionality already works.

Raspberry Pi receives commands via wi-fi and connected to Tank hardware using GPIO.
Perl script is used to read key presses and send right commands to GPIO. Controls are ‘W’ ‘A’ ‘S’ ‘D’ for movement and ‘[‘ ‘]’ for moving tower left and right.
