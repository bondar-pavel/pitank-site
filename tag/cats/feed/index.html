My PiTank was collecting dust under the table for a while and today I got some free time to bring it back to life. There were some issues with connecting over ssh, so I connected it to my desktop monitor with hdmi cable and start debugging. When I got all services up and running cat’s attention was attracted by tank and I just started to record what happened.

By the way, since last videos PiTank was upgraded and you can see now bluetooth speaker on the back of PiTank. It is connected using 3.5″ jack to Raspberry Pi analog output and can be used to reproduce different sounds or just play music. For now I just run ‘espeak’ with some text, so it reads text from console.

]]> Pitank could have a lot of practical usages.  But live is strange thing, so I basically use it to stream cat videos…

When cat is left alone at home and I am somewhere far away, I just login to my pitank and start watching what is happening.

Sometimes I stream it directly to youtube.  So here is one of the first such streaming sessions:

By the way streaming to youtube from Raspberry pi is as simple as one line command:

raspivid -o – -t 0 -fps 30 -b 6000000 | ffmpeg -re -ar 44100 -ac 2 -acodec pcm_s16le -f s16le -ac 2 -i /dev/zero -f h264 -i – -vcodec copy -acodec aac -ab 128k -g 50 -strict experimental -f flv rtmp://a.rtmp.youtube.com/live2/your-streaming-key

raspivid sends video stream from picamera to sdtout, it is already encoded by h264. ffmpeg puts this video stream in flv container and sends to youtube. No video encoding/decoding  happens, so it works pretty good.  Also ffmpeg emulates sound track to make stream acceptable by youtube.

raspvid should be available in all recent distributions suitable for Raspberry Pi. And ffmpeg is not present on some distributions, so compiling from binaries might be required.

]]> Once upon a time PiTank project visited fresh air and met new friends.

We went to the forest, drove through the city streets and turned turtle on the leaf litter. In the forest we met nice cat. I guess it was his territory, so the alien (pitank) got a lot of attention.

A few words about PiTank project.

PiTank – is a Raspberry Pi based remotely controllable tank. It can be controlled from web browser using keyboard or mouse. Video from camera module mounted on top of PiTank is streamed directly into the browser. Technically PiTank can be remotely controlled from any point in the world where you have internet connection.
