By the way streaming to youtube from Raspberry pi is as simple as one line command:

raspivid -o – -t 0 -fps 30 -b 6000000 | ffmpeg -re -ar 44100 -ac 2 -acodec pcm_s16le -f s16le -ac 2 -i /dev/zero -f h264 -i – -vcodec copy -acodec aac -ab 128k -g 50 -strict experimental -f flv rtmp://a.rtmp.youtube.com/live2/your-streaming-key

raspivid sends video stream from picamera to sdtout, it is already encoded by h264. ffmpeg puts this video stream in flv container and sends to youtube.Â No video encoding/decoding Â happens, so it works pretty good.Â  Also ffmpeg emulates sound track to make stream acceptable by youtube.

raspvid should be available in all recent distributions suitable for Raspberry Pi.Â And ffmpeg isÂ not present on some distributions, so compiling from binaries might be required.
