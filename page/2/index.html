Raspberry Pi powered Tank

This project sets Raspberry Pi on wheels (tracks in fact), allows to control tank remotely and get streamed video over internet.

Based on Raspberry Pi

We use Raspberry Pi as a computing platform. It has enough computing power, GPIO pins for controlling tank hardware and pretty compact size.

Stream video from PiTank

Tank has camera on the top of the tower and it can be rotated in 2 axis.

Open Source

All code is on github, so feel free to improve and reuse it!
